# Single Line Comment

Flow-Wing supports single line comments. They start with `/;`.

### Example Usage:

<CodeBlock code={
`/; This is a single line comment
print("Hello, World!")
`} language="fg"/>

**Output:**
<CodeBlock code={
`Hello, World!
`} />
